Beispielressourcen für das Muster Service Facade to WebSphere MQ: one-way with acknowledgment (Servicefassade für WebSphere MQ: unidirektional mit Empfangsbestätigung)

Das Muster Service Facade to WebSphere MQ: one-way with acknowledgment (Servicefassade für WebSphere MQ: unidirektional mit Empfangsbestätigung) bietet eine Reihe von Beispielressourcen. Zu diesen Beispielressourcen gehören ein Nachrichtensatz und eine implementierbare Web Services Description Language (WSDL). Mithilfe der Beispielressourcen können Sie diese Muster ganz einfach nutzen.

Hinweis: Die Beispielressourcen können nur über die Links im Information Center importiert und entfernt werden, wenn das in IBM Integration Toolkit integrierte Information Center verwendet wird.

Sie können die Beispielressourcen einrichten, indem Sie auf folgenden Link klicken:

  • Beispielressourcen in den Arbeitsbereich importieren

Speichern Sie die Datei OrderService.wsdl, um die ursprüngliche WSDL-Datei für Tests mit Clientanwendungen abzurufen.

Wenn Sie die Nutzung der Beispielressourcen abgeschlossen haben, können Sie diese entfernen; klicken Sie hierfür auf folgenden Link:

  • Beispielressourcen aus dem Arbeitsbereich entfernen
